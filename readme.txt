# 以下是池子实测启动项目的流程
2、确保项目jdk环境为1.8
3、安装neo4j
4、项目启动程序：
   2.1、配置application.yml 中的mysql中的库名、访问密码，并执行sql脚本knowledgegraph.sql；
   2.2、配置application.yml 中的neo4j的密码；
5、启动项目，访问地址：http://localhost:8080/kg/index
